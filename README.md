## Консольный Банк на Java

### Описание
Учебный проект.
Консольное приложение, которое моделирует работу банка: создание счетов, операции, отчёты и начисления.

После запуска выводится меню:
```
1.	Создать клиента
2.	Открыть дебетовый счёт
3.	Открыть кредитный счёт
4.	Пополнить
5.	Снять
6.	Перевести
7.	Показать счета клиента
8.	Показать транзакции
9.	Отчёт банка
0.	Выход
Введите номер задачи и нажмите Enter:

```
Работа приложения осуществляется выбором нужного пункта меню путем ввода соответствующей цифры и нажатия Enter после приглашения программы.
Далее приложение просит ввести необходимые сведения и ждет ввода с последующим нажатием Enter.
Далее приложение выводит сообщения об успешной/неуспешной операции.

Транзакции сохраняются и могут быть выведены в консоль в п. 8 меню.

Отчет банка в виде:
o	сколько счетов каждого типа
o	суммарные балансы по типам
o	количество успешных/неуспешных операций
выводится в п. 9 меню.

При создании счетов в начале номера счета добавляется префикс "00", устанавливается переменной "accountNumberPrefix" в классе Bank.

Для выхода из приложения в основном меню нужно ввести 0 и нажать Enter.
Для аварийного завершения работы необходимо нажать клавиши Ctrl + C.
